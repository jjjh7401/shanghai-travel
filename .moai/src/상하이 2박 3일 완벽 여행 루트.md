### 🚕 디디(DiDi) 택시 주의사항 3가지
1. **이헤일링(앱 호출) 구역**에서 호출할 것
2. **하차 후 결제 화면에서 톨비·주차비 추가 확인!** → 동의 안 했다면 '헬프' 눌러 상담원 연결 → 삭제 요청 가능
3. **냄새 걱정된다면 프리미어 호출** → 깔끔한 모범택시 수준, 생수 제공

## 📅 Day 1 — 도착 후 시내 감성 산책 코스

```
공항 → 숙소 체크인 → 점도덕 → 아마수작 → 임시정부 
→ 우캉루 → 용캉루 → 안푸루 → 난징시루·장원 → 라오지우예 → 와이탄 야경
```

### 🍽️ Day 1 맛집 & 카페

**🥟 점도덕(點都德) — 딤섬**
- 한국인들도 많이 찾는 유명 딤섬 맛집
- 추천 메뉴: **호미 창펑 + 하가오**
- 숙소에서 도보 약 10분

**🧋 아마수작 — 밀크티**
- 웨이처 카드로 미리 사전 주문 추천
- 팥 곡물 + 팔떡 모찌 식감, 포만감 굿

**🥘 동북사계교자왕 — 꿔바로·교자**
- 우캉맨션 주변 동북 음식 전문점
- 맥주 + 꿔바로 + 토마토 계란볶음 = **94위안**

**🍲 라오지우예(老九爷) — 훠궈**
- 현지인 훠궈 맛집
- 백탕(오리뼈) / 홍탕(마라) 선택
- ⚠️ **주의: 백탕 잘못 고르면 개구리 탕이 될 수 있음! 꼭 확인**
- 새우 메뉴 강력 추천
- 2인 기준 **356위안**

### 🌃 야경 포인트
- **와이탄(더 번드)** 야경 감상

## 📅 Day 2 — 루자주이·황푸강·북와이탄 코스

```
루자주이 → 샤오양셩지엔 → 루이싱커피 → 상하이타워(도운서점) 
→ 디즈니스토어·샤오미 → 황푸강 페리 → 리바이시에 → 북와이탄 
→ 허마선생 → 하프어드링크 → 상하이박물관 → 황푸공원 야경
```

### 🍽️ Day 2 맛집 & 카페

**🥟 샤오양셩지엔 — 상하이식 성지엔(군만두)**
- 새우 / 고기 / 게살 세 종류
- 세 종류 합산 약 **104위안**

**☕ 루이싱커피(Luckin Coffee)**
- 자스민 티 라떼, 코코넛 라떼 추천

**📚 상하이타워 52층 도운서점 ⭐ 꿀팁**
- 전망대 입장권 **180위안(약 35,000원) 대신 무료!**
- 사전 위챗 예약 필수 → 무료 입장
- 단, 뷰 구역 입장 시 **1인 1메뉴 구매 필수**

**⛵ 황푸강 페리 ⭐ 꿀팁**
- 수십만원짜리 유람선 대신 **단돈 2위안!**
- 낮에 한 번 + 밤에 한 번 추천
- 동방명주·와이탄 뷰 그대로 감상 가능

**🦀 리바이시에 — 게살국수**
- 현지인 맛집, 번호표 대기 약 20분
- 게살국수 + 맥주 = **202위안**

**🐛 허마선생(Hema) — 슈퍼마켓**
- 따룬파(대형마트) 대신 추천
- 하마 캐릭터 기념품 상품 풍부
- 지도에서 근처 지점 검색 후 방문

**🍢 하프어드링크 — 양꼬치**
- 인민광장 인근 로컬 양꼬치 맛집

### 🌃 Day 2 야경 포인트 ⭐ 강력 추천
> **와이탄보다 북와이탄 + 황푸공원을 추천!**
> - 사람이 훨씬 적고 동방명주 뷰가 더 멋짐
> - 황푸공원에서 동방명주 배경 인증샷 필수!

## 📅 Day 3 — 브런치·티엔즈팡·기념품·출국

```
지엔빙(아침) → 매너커피 → 릴리안 베이커리 → 패왕차희 
→ 티엔즈팡(기념품 쇼핑) → 마마지아(점심) → 젤네일 → 공항
```

### 🍽️ Day 3 맛집 & 카페

**🫓 지엔빙(煎饼) — 중국식 길거리 크레페**
- 숙소 근처에서 아침으로 간단하게
- 감자채 추가 추천!

**☕ 매너커피(Manner Coffee)**
- 코코넛 라떼 강력 추천
- 테이크아웃 후 티엔즈팡 산책 코스

**🥧 릴리안 베이커리 — 에그타르트**
- 역사 약 20년의 전통 에그타르트 명가
- 티엔즈팡 쇼핑몰 인근 위치

**🧋 패왕차희 — 밀크티**

**🍛 마마지아 — 현지 가정식**
- 마파두부 ⭐ 특히 강력 추천 (한국과 다른 찐 맛!)
- 홍소육 + 어양가지(가지볶음) 함께 추천

### 🛍️ 기념품 쇼핑 — 티엔즈팡
| 추천 아이템 | 설명 |
|------------|------|
| 칭즈 핸드크림 | 현지 인기 브랜드 |
| 블렌버니 | 상하이 로컬 브랜드 |
| 따바이투 | 전통 과자·캔디 |
| 하마 캐릭터 상품 | 허마선생 굿즈 |

### 💅 젤네일 팁
- 따종(大众点评) 앱에서 첫 구매 할인 적용
- 약 **150위안** 수준으로 가능
